# Masajes Majo - Página Web

## Cómo subirla a GitHub Pages (Hosting gratuito)

1. Crea una cuenta en [GitHub.com](https://github.com)
2. Crea un nuevo repositorio llamado `MasajesMajo`
3. Sube el archivo `index.html` (y la carpeta `img/` si agregas fotos)
4. En el repositorio, ve a **Settings → Pages**
5. En “Build and deployment”, selecciona:
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** /(root)
6. Guarda y espera unos segundos.
7. Tu sitio estará disponible en una dirección como:
   **https://tuusuario.github.io/MasajesMajo/**

Disfruta tu web profesional 💆‍♀️
